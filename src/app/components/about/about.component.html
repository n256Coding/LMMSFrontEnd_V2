About LMMS

LMMS is an online tool designed for lecturers to find online resources and check their lecture material with e-learning standards.

Finding resources to define the syllabus content and, creating and maintaining the lecture materials according to the learning standards or organization standards, can be a challenging task in universities. As of yet, no system has been implemented to overcome this overall problem.

We will provide an advanced searching area to suggest resources to the lecturers according to their exact requirement that will help to define their syllabus content. We will drastically reduce a lot of issues that lecturers face in their university life, and it saves more time for lecturers. It also helps to maintain standards within the organization for those who will use this system.

How Lecture Material Management System works

“Lecturer Material Management System” is a unique system specially made for academics in universities and other educational institutes. The system facilitates a university’s Quality Assurance Department to set standard templates for lecture material (Lecture Slides, Lab Sheets, and Tutorials) and also the Learning Management System (LMS) page. Individual lecturers can use “Lecturer Material Management System” to check for compliance to university standards. In addition, a web based advanced search option allows academics to easily find learning resources that match specific requirements the lecturer needs in addition to meeting the level of the students The university can easily customize the standard templates according to university standards. These include using Master Pages with University logos and specific themes the Power Point slides and the use of specific headers and footers in the Lab and Tutorial Sheets.

“Lecturer Material Management System” allows Universities to easily maintain quality assurance standards of academic content developed and shared in a University. This would enable consistency of all a cademic material produced by the University which will directly enhance the student’s perception of the University.

Services

Textual Resources Searching Service

This system has results caching mechanism to speedup results. But if user requests some results that not cached yet, system should perform non-repetitive operation to collect information from internet and analyze. As a result, it will take more time to do that extra operations. The time that taken to collect and analyze will depend on requested resource type. If it is web pages related contents, it would take approx. 5 minutes. If it is a PDF related content (E-Books), because of the large amount of content and file size, it would take approx. 15 - 25 minutes. But it is depending on file size of identified E-Books

Following steps describes the how this service works.

  • Go to Text Analyzer Tab.
  • Enter search query “SQL database operations” in search box
  • Select “web site” option from material type
  • Click search button
  • It should take some time to load and analyze content and will show results when completed.

Sample Keywords to try

  • SQL Database operations.
  • Java arrays
  • Java oop concepts
  • Html tags
  • Android layout designing
  • Java Threading

Video Resources Finding Service

Video finder and analyzer helps you to identify tutorial videos under the programming domain. So you can identify best matching tutorial videos that suitable for your students.

Following steps describes the how this service works.

  • Log in as a lecturer.
  • Navigate to the video analyzer page.
  • Enter your search keyword. (ex: “Java threading”, “android listviews”, “OOP concepts”)
  • Select filters.
  • Click the search button
  • Can’t find the video that you are looking for? Use Start new search button to start a new search for you. This feature will take some time to complete. Once processing completed you will get an email that contains a link to your search results.

Note: start a new search and analyzing function is currently under development.

Slide Validation Service

Slide Validator helps you to identify consistency between your lecture material and university standards by considering learning standards.

Note: This system already has two template files for both PowerPoint slides and Word documents.

Please use these files for validations. (uploaded templates- https://drive.google.com/open?id=16X8iPwjZ7VrO0p44RDtRm2YWBB7dZvym)

Steps to validate PowerPoint slides

  • Log in as a lecturer.
  • Navigate to the 'Slide Validation' feature.
  • Choose 'Slide' as your lecture material type.
  • Choose 'Slide Template' as your template.
  • Choose your lecture slide and click the 'Upload' button. (We have already provided set of lecture slides in below section. Please download them and use those file for validations)
  • Choose your validation type ('Standard Template' or 'Customizable Template')
  • Provide required information based on the selected validation type.
  • Click 'Start Validation' button to start the validation process.
  • Click 'Generate Report' button to get violation report.

Samples: https://drive.google.com/open?id=1gzUL5sS8Gz6Dik2NBiWnvD8PJt528V6x

Steps to validate Word documents

  • Log in as a lecturer.
  • Navigate to the 'Slide Validation' feature.
  • Choose 'Tutorial' or 'Lab Sheet' as your lecture material type.
  • Choose 'Word Template' as your template.
  • Choose your lecture tutorial or lab sheet and click the 'Upload' button. (We have already provided set of Word files in below section. Please download them and use those file for validations)
  • Choose your validation type ('Standard Template' or 'Customizable Template')
  • Provide required information based on the selected validation type.
  • Click 'Start Validation' button to start the validation process
  • Click 'Generate Report' button to get violation report

Samples: https://drive.google.com/open?id=1xe6TkskjhKm2WSfVsJdQvOWm_u8odm-j

Moodle Page Validation Service

Moodle validation Service allows you to check your course page consist with learning standards and provide options to customize it. Moodle Quiz analyzing Service provide lecturer to check his moodle quiz file(XML) is according to

Steps to validate Moodle Course page

  • Login to the site.
  • Go to the Moodle Validation Tab.
  • Select "Moodle Page Validation" option.
  • Provide the creadentials and select the Validation Option. (We do not store any credentials of yours.)
  • If you select "Standard Validation" as your option, then select "Analyze" button.
  • It you select "Custom Validation" as your option, then fill the required fields and start analyzing process.

Note: Standard validation process done according to identified learning standards.
In "Custom validation" user can choose which standard should include in thr process or not.

Steps to validate Moodle Quiz file

  • Login to the site.
  • Select the Moodle Quiz file validation Tab.
  • Upload your quiz file. (File should be in Moodle XML format. Otherwise process will not work correctly.)
  • Choose analyzing option.
  • If you select "Basic Analyzing", then click start button.
  • It you select "Custom Analyzing", then fill the required fields(ex: no of question : 50)
  • Start the process.

Note : Analyzing process might get some time to display the results.
